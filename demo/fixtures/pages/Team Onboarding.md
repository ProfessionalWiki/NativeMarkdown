# Team onboarding

Welcome aboard! This page gets you productive in your first week. It is written in Markdown — hit edit to
see how little markup there is.

## Before you start

Make sure IT has set you up with:

- [ ] Laptop and VPN access
- [ ] Git repository permissions
- [ ] An account on this wiki

[[Deployment Guide.md]]. Questions go to the `#onboarding` channel.
